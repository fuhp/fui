CheckboxGroup

FCheckboxGroup管理多个checkbox组件，将被勾选的复选框的value值拼接为逗号分隔的字符串作为表单的提交值。

依赖组件

  • 无

适用范围

功能类似于选项不多且明确的复选combo，是另一种的展现方式

约束

  1. FCheckboxGroup不管理内部checkbox布局

注意事项

  1. 内部包含的checkbox不能有name属性，但必须有value值

属性

defaultValue String

组件的默认值，在组件初始化或者调用reset的时候，会采用此默认值初始化checkbox的选中状态， 比如defaultValue="0,1,3"，那么value为0,1,3的checkbox会被选中。注意：defaultValue的值分隔串必须与seperator属性一致。
