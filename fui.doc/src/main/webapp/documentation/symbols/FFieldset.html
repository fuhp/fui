Fieldset

FFieldset和html的<fieldset>功能类似，但是在其基础上进行了扩展，统一了样式，支持折叠和展开。

依赖组件

  • 无

适用范围

一般用于栏目分类分组，方便用户查看，并支持隐藏相关内容，节省可视空间。

约束

无

注意事项

无

autoScroll Boolean

如果设置为true，表示控件在必要时自动显示横向或纵向滚动条。默认值为true。 (仅标签使用)


示例： 
无
 true
collapsed Boolean

如果为true，将FFieldset渲染成收缩的，否则渲染成展开的(默认值为false)。(仅标签使用)
如果设置了此值为true，并且同时设置了collapsible为false，那么此值设置无效，
请慎重设置此值：如果在组件初始化设置了此值为true，那么在Fieldset第一次展开的时候在部分浏览器下会造成组件的宽（高）度有调整，但是之后将会表现正常。
