Message

替换原生Javascript的alert()，避免浏览器被阻塞的问题。

依赖组件

  • FMask

适用范围

Message用于提供提示信息的弹出窗口，使用alert()、confirm()等函数时出现的那种提示信息的弹出窗口。

约束

  • 只能作为简要消息提示，不能用作fwin等复杂的场景中；

注意事项

  • 无

方法

名称 参数 描述
alert (title, msg, callback)

alert消息框

参数:
  • :title 消息的标题
  • :msg 消息的内容
  • :callback 消息关闭的时候的回调方法
    • 示例： 
    $.FMessage.alert("Attention", "Everything is ok!");
    confirm (title, msg, callback)

    confirm消息框

    参数:
  • :title 消息的标题
  • :msg 消息的内容
  • :callback 消息关闭的时候的回调方法
    • 示例： 
    function showResult(btn) {
  alert('You clicked the "' + btn +'" button');
};
$.FMessage.confirm('Confirm', 'Are you sure you want to do that?', showResult);
    prompt (title, msg, callback)

    带一个输入框的消息框

    参数:
  • :title 消息的标题
  • :msg 消息的内容
  • :callback 消息关闭的时候的回调方法
    • 示例： 
    function showResultText(btn, text){
  alert('You clicked the "' + btn +'" button and entered the text "' + text + '"');
};
$.FMessage.prompt('Confirm', 'Are you sure you want to do that?', showResultText);
    show (op)

    弹出指定参数的消息框

    参数:
  • :op 消息框的弹出选项，可以有下列属性：
    1. title: 消息框标题；
    2. msg: 消息的内容，对于prompt，就是输入框的提示信息；
    3. callback: 输入框在关闭的时候出发的回调方法，非必须；
    4. width: 输入框的宽度，默认情况下自动设置合适的宽度，也可以通过设置此属性修改；
    5. icon: msg前的图标，目前只支持 $.FMessage.INFO、$.FMessage.QUESTION、 $.FMessage.ERROR、$.FMessage.WARNING；如果没有设置此属性，则不显示图标；
    6. prompt: boolean类型，true则表示显示一个带输入框的消息框，此时icon属性无效；
    7. buttons: Message显示的按钮 
      $.FMessage.OK 只显示一个"OK"按钮
$.FMessage.OKCANCEL 显示一个"OK"按钮，一个"Cancel"按钮
$.FMessage.YES 只显示一个"Yes"按钮
$.FMessage.NO 只显示一个"No"按钮
$.FMessage.YESNO 只显示一个"Yes"按钮，一个"No"按钮
$.FMessage.CANCEL 只显示一个"Cancel"按钮
$.FMessage.YESNOCANCEL 显示"Yes"、"No"、"Cancel"按钮

这些按钮可以自由组合，比如 $.FMessage.OK|$.FMessage.CANCEL 效果与 $.FMessage.OKCANCEL一致
    8. buttonTexts: 配置按钮对应文字 
       $.FMessage.show({
   title: 'Icon Support',
   msg: 'Here is a message with an icon!',
   buttons:  $.FMessage.OK | $.FMessage.YESNOCANCEL,
   icon: $.FMessage.QUESTION,
   buttonTexts : {
     yes : "I say yes!",
     no : "No more...",
     cancel : "Bye."
   }
 });
    9. closeIcon 是否显示右上角的关闭按钮，false则不显示按钮，默认显示
