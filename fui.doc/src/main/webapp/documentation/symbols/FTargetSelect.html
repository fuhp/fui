TargetSelect

不含有隐藏域的输入框，在默认情况下是只读的，可以绑定点击图标的回调。

依赖组件

  • 无

适用范围

一般应用于点击按钮弹出一个FWin， 然后将在FWin上操作的结果回填到输入框中。

约束

  • 无

注意事项

  • 无

使用示例

点击按钮弹出win，并将选中的值回填

								setValue
								(v, force)
								
									
    设置表单输入框的值，如果传入了参数则表示设置；该方法会始终返回当前表单输入框的值；如果存在隐藏域则访问隐藏域，否则效果同方法displayValue(v)。
    
注意：不建议绑定输入框的 change或者onChange事件，因为这些事件
在通过js修改输入框的value之后不会触发（很奇妙的是，在IE6/7/8上居然触发了），所以在这种情况下请绑定 onValueChanged事件。
    
另外在默认情况下，调用此方法对输入框赋值，会触发onValueChanged事件，可以通过传入force=true来关闭事件的触发。
