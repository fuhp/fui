Ajax

在需要进行点击交互（如数据提交等），可以通过button来作为事件触发的载体，统一button和button在toolbar中的展现样式。

依赖组件

  • 无

适用范围

适用于需要点击交互的场景，比如表单提交、操作取消、弹出win等等

约束

  1. 默认不提供button按钮hover和click的样式变化（主要是性能的考虑）；
  2. 提供了比较常用onClick的事件，其他的事件（hover、mousedown等）没有提供， 项目组可以根据需要使用jquery绑定，比如$("#button1").mousedown(function() {/* do something */})；
  3. button只支持文本前有图标，并且没有提供API在初始化完毕之后修改或者去除图标；

方法

名称 参数 描述
getList (options)

获取应答中的列表对象，FAjax请求的数据类型需要为list、listsimple、tree，在success回调中的data参数为Array类型。
用户请求不同的数据类型时，需要在data参数中传递_respType参数，_respType参数需要指定需要的数据类型。类型有：list，listsimple，tree。

参数:
  • :options FAjax的配置参数，结构是{url:服务,data:{请求参数},success:function(data，textStatus, jqXHR){//returnCode==0的情况},failure:function(data,textStatus, jqXHR){//returnCode为 1或者 -1的情况 },error:function(jqXHR, textStatus, errorThrown){//请求错误时，例如请求超时}}
    • 示例： 
    $.FAjax.getList({
             url:'com.hundusn.user.list.fservice',
             data:params,
             success:function(data,textStatus, jqXHR){
                 //请求正常，returnCode ==0 时触发
             },
             failure:function(data,textStatus, jqXHR){
                 //请求正常，returnCode !=0时触发
             },
             error : function(jqXHR, textStatus, errorThrown){
                 //请求失败时错发，例如请求超时。
             }
             );
    getObject (options)

    获取应答中的对象，FAjax请求的数据类型需要为pojo和page，在success回调中的data参数为object类型。
    用户请求不同的数据类型时，需要在data参数中传递_respType参数，_respType参数需要指定需要的数据类型。类型有：pojo，page。

    参数:

    remote (options)

    统一的AJAX操作接口,如果FAjax设置了_errorHandlerMap或者_defaultErrorHandler，failure回调函数的返回值不為true，则不执行默认的错误处理函数，认为用户自行处理错误处理。
    用户请求不同的数据类型时，需要在data参数中传递_respType参数，_respType参数需要指定需要的数据类型。类型有：list，page等。 另外，还支持的参数有：

    1. _respType : 应答的数据模型，支持list、listsimple、pojo、page、tree和jres，其中数据模型“jres”支持多数据集IDataset；
    2. _respMapping : 应答结果字段的映射关系，比如返回的结果集有字段“userId”和“upperId”，但是需要的是“id”和“pid”， 那么可以设置此值为"id=userId,pid=upperId"，每个字段使用英文逗号","分隔，FUI的Servlet会自动做字段转换。

    参数:
  • :options FAjax的配置参数，结构是{url:服务,data:{请求参数},success:function(data，textStatus, jqXHR){//returnCode==0的情况},failure:function(data,textStatus, jqXHR){//returnCode为 1或者 -1的情况 },error:function(jqXHR, textStatus, errorThrown){//请求错误时，例如请求超时}}
    • 示例： 
    $.FAjax.remote({
            url:"com.hundsun.user.add",
            data:$('#formId').getValues(),
         success:function(data,textStatus, jqXHR){
             //请求正常，returnCode ==0 时触发
          },
         failure:function(data,textStatus, jqXHR){
              //请求正常，returnCode !=0时触发
         },
         error : function(jqXHR, textStatus, errorThrown){
             //请求失败时错发，例如请求超时。
         }
         //其他一些参数。
         );
