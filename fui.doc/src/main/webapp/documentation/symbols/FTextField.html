TextField

统一原生html的 <input type="text"> 的样式和风格，不含有隐藏域

  • 无

适用范围

适用于form中绝大部分 <input type="text"> 的使用场景。

方法

名称 参数 描述
displayValue (v, force)

设置/获取显示域的值，如果传入了参数则表示设置；该方法会始终返回当前显示域的值。
注意：不建议绑定输入框的 change或者onChange事件，因为这些事件 在通过js修改输入框的value之后不会触发（很奇妙的是，在IE6/7/8上居然触发了），所以在这种情况下请绑定 onValueChanged事件。
另外在默认情况下，调用此方法对输入框赋值，会触发onValueChanged事件，可以通过传入force=true来关闭事件的触发。

参数:
  • :v 要设置的显示域的值
  • :force 非必须，如果传入true则不会触发onValueChanged事件
