Button

在需要进行点击交互（如数据提交等），可以通过button来作为事件触发的载体，统一button和button在toolbar中的展现样式。

button在toolbar中：

依赖组件

  • 无

适用范围

适用于需要点击交互的场景，比如表单提交、操作取消、弹出win等等

约束

  1. 默认不提供button按钮hover和click的样式变化（主要是性能的考虑）；
  2. 提供了比较常用onClick的事件，其他的事件（hover、mousedown等）没有提供， 项目组可以根据需要使用jquery绑定，比如$("#button1").mousedown(function() {/* do something */})；
  3. button只支持文本前有图标，并且没有提供API在初始化完毕之后修改或者去除图标；

注意事项
