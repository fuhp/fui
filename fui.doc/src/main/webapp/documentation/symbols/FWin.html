Win

用于弹窗方式进行数据展示或交互，他是一个弹出式容器，但是不管内容的布局。

依赖组件

  • 无

适用范围

适用于弹窗式数据展示或者交互方式，典型的如增加，修改，查看的页面。

约束

  • 无

注意事项

  • 无

名称 类型 描述 默认值
buttonAlign String

窗体中的按钮的对齐方式 center left right

	        					buttons
	        					Array
	        					
	        						
窗体中的按钮。此配置项为JSON数组。每个JSON对象具有 text 属性(配置按钮文字)
、 click 属性(配置按钮触发时的回调方法)、iconCls属性（配置按钮的样式图片）。其中iconCls的样式如下所示：
.iconCls {background: url("../dd/drop-yes.gif") no-repeat scroll left center transparent;}

		        					
		        						

		        						示例：
		        						
		        							
  $("#select").FWin({buttons : [{
     text : "确定",
     iconCls:"isOk",
     click : function () {...}
 }, {
     text : "取消",
     iconCls:"cancel",
     click : function () {...}
 }]);

		        						
		        					
	        							
	        					[]
	        				

	        			
	        				

	        					draggable
	        					Boolean
	        					
	        						
设置组件是否可拖动。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					true
	        				

	        			
	        				

	        					hasCloseBtn
	        					Boolean
	        					
	        						
设置win组件右上角是否有关闭按钮，默认值为true，即有关闭按钮。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					true
	        				

	        			
	        				

	        					height
	        					Number
	        					
	        						
组件的高度。

		        					
		        						

		        						示例：
		        						
		        							
  $("#select").FWin({height : 300});

		        						
		        					
	        							
	        					300
	        				

	        			
	        				

	        					isIframe
	        					Boolean
	        					
	        						
是否以iframe的方式加载页面,如果该属性值为true，生成的iframe标签的id为“FWin组件的id-iframe”。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					false
	        				

	        			
	        				

	        					maxable
	        					Boolean
	        					
	        						
是否可最大化。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					false
	        				

	        			
	        				

	        					maximized
	        					Boolean
	        					
	        						
设置win组件打开时是否最大化窗口。默认值为false。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					false
	        				

	        			
	        				

	        					modal
	        					Boolean
	        					
	        						
是否模态窗口。默认值为true，即以模态的方式展现窗口。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					true
	        				

	        			
	        				

	        					pageUrl
	        					String
	        					
	        						
组件加载页面的地址,如果isIframe属性为true,那么请求的页面需要是标准的HTML页面，如果isIframe属性为false，那么请求的页面只能是HTML片段，不允许包含html、head、body这些html标签。

		        					
		        						

		        						示例：
		        						
		        							

		        						
		        					
	        							
	        					""

窗体的标题。

方法

            	

          			
          				

          					名称
							参数
							描述
          				

          				
          					

								close
								()
								
									
关闭窗体。

									
									
	                               		示例：
	                                    
	                                        
$("#select").FWin('close');

	                                    
	                           		 
										
							

          				
          					

								setHtml
								()
								
									
设置FWin组件内容区域的html片段。

									
									
	                               		示例：
	                                    
	                                        
 $("#select").FWin('setHtml','<div>我是content内容</div>');

	                                    
	                           		 
										
							

          				
          					

								setPageUrl
								(pageUrl)
								
									
设置FWin组件请求页面的路径，如果isIframe属性为true，则以iframe的方式加载页面。否则以ajax的方式获取内容html，然后添加到FWin组件中。注意，调用setPageUrl方法之前，需要先调用show方法。

									
										参数:
		                                
		                                    
  • 
		                                        :pageUrl
		                                        请求页面的路径
		                                    
    • 
		                                
									
									
	                               		示例：
	                                    
	                                        
     $("#select").FWin('setPageUrl','contentPage.jsp');
    
	                                    
	                           		 
								    		
							
    
          				
          					
    
								setSize
								(w, h)
								
									
    通过传入的参数，设置组件的高宽并重新居中显示。注意事项：调用该方法之前，必须先调用组件的show方法展现组件。
    
									
										参数:
		                                
		                                    
  • 
		                                        :w
		                                        组件的宽度（number类型）
		                                    
    • 
		                                
		                                    
  • 
		                                        :h
		                                        组件的高度(number类型)
		                                    
    • 
		                                
									
									
	                               		示例：
	                                    
	                                        
    $("#select").FWin('setSize',w,h);
    
	                                    
	                           		 
								    		
							
    
          				
          					
    
								show
								()
								
									
    打开window窗体。
    
									
									
	                               		示例：
	                                    
	                                        
     $("#select").FWin('show');
