ButtonGroup

自动撑开父元素的宽度，将button居中、居左或者居右排布。

依赖组件

  • 无

适用范围

适用于panel、Fieldset和win等容器中排布button，并独占一行。

约束

  1. buttonGroup容器没有提供边框，项目组可以根据需要设置边框样式
  2. 独占一行，宽度撑满父容器
