Tree

支持动态数据（ajax异步树）加载和静态数据两种数据加载模式；支持单选和复选；可以绑定树节点点击事件

依赖组件

  • FAjax

适用范围

适用于主页面的菜单树，也适用于需要类似于组织机构展现方式。

约束

  • 不支持后台组装树的DOM，即dom结构是在浏览器端生成；
  • 仅支持异步树，并且每次最多展开一级；

注意事项

  • 无
